Detailed Description

template<typename TInputItem, typename TOutputItem = TInputItem>
class CWorker< TInputItem, TOutputItem >

Implements IWorker to process items from an input queue and place results in an output queue, using a dedicated worker thread.

Template Parameters
TInputItemThe type of items in the input queue.
TOutputItemThe type of items in the output queue (defaults to TInputItem).
