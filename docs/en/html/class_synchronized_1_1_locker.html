Detailed Description

template<typename TMutex, bool const is_tracable = false>
class Synchronized< TMutex, is_tracable >::Locker

A scoped locker class that acquires the Synchronized object's mutex on construction and releases it on destruction. Also provides methods for waiting on a condition variable. Inherits from NonCopyable and NonMovable to prevent accidental misuse.
