Detailed Description

template<typename TEvent>
class iridium::pattern::IObservable< TEvent >

Interface for the 'Observable' (or 'Subject') part of the Observer design pattern. Objects implementing this interface can be observed by IObserver instances. When an event of type TEvent occurs, the observable is responsible for notifying its subscribed observers.

Template Parameters
TEventThe type of event data that observers will be notified about. This defines the context of notifications.
