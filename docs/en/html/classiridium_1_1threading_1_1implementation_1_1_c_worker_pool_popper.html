CWorkerPoolPopper< TItem >::TItems iridium::threading::implementation::CWorkerPoolPopper< TItem >::pop ( bool const & is_do_wait)
overridevirtual

Pops items from the queue. Retrieves one or more items from the front of the queue. The behavior regarding waiting depends on the implementation and the is_do_wait flag.

Parameters
is_do_waitIf true, the method may block until items are available. If false, it may return immediately if the queue is empty.
Returns
A list of items popped from the queue. The list might be empty depending on is_do_wait and queue state.

Implements iridium::threading::IAsyncQueuePopper< TItem >.
