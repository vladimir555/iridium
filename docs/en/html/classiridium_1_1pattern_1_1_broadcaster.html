Detailed Description

template<typename TReceiver>
class iridium::pattern::Broadcaster< TReceiver >

Implements the Broadcaster pattern, allowing messages to be sent to multiple receivers.

This class manages a collection of receivers and provides a mechanism to invoke a specific member function on all of them.
