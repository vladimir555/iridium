iridium::parsing::serialization Namespace Reference

Provides classes, macros, and utilities for defining serializable/deserializable data structures. This namespace focuses on a node-based representation where data structures are defined as a hierarchy of nodes, each potentially having attributes, values, and child nodes or lists of nodes. The system uses NodeView classes as wrappers around an underlying INode tree to provide typed access and modification, and employs a suite of macros (DEFINE_ROOT_NODE_BEGIN, DEFINE_ATTRIBUTE, etc.) to declaratively define these structures. Detailed Description

Provides classes, macros, and utilities for defining serializable/deserializable data structures. This namespace focuses on a node-based representation where data structures are defined as a hierarchy of nodes, each potentially having attributes, values, and child nodes or lists of nodes. The system uses NodeView classes as wrappers around an underlying INode tree to provide typed access and modification, and employs a suite of macros (DEFINE_ROOT_NODE_BEGIN, DEFINE_ATTRIBUTE, etc.) to declaratively define these structures.
