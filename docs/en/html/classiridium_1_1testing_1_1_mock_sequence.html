Detailed Description

template<typename TClassMock>
class iridium::testing::MockSequence< TClassMock >

Forward declaration for the MockSequence class template.

Manages a sequence of expected method calls on a mock object. This class allows defining a strict order in which methods are expected to be called, along with specific arguments for those calls.

Template Parameters
TClassMockThe type of the mock class (e.g., MyInterfaceMock) for which the sequence is defined.
