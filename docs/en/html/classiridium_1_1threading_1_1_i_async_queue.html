Detailed Description

template<typename TItem>
class iridium::threading::IAsyncQueue< TItem >

Interface for an asynchronous queue providing both push and pop operations. This interface combines IAsyncQueuePusher and IAsyncQueuePopper and adds functionality for queue interruption and state checking.
