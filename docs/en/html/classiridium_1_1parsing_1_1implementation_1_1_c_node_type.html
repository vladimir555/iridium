Detailed Description

template<typename TValue>
class iridium::parsing::implementation::CNodeType< TValue >

Concrete implementation of the INodeType interface. This class template provides a full implementation for a node in a tree-like structure. It manages the node's name (using string interning for efficiency), its primary value of type TValue, and a list of child nodes. It is designed to be cloneable via the IPrototype interface.

Template Parameters
TValueThe type of the primary value stored in the node.
