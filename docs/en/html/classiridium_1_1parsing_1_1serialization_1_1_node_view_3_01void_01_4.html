Detailed Description

Specialization of NodeView for typeless nodes (void value type). This is typically used as a base class for structured nodes defined by DEFINE_NODE_BEGIN or DEFINE_ROOT_NODE_BEGIN, or to represent a parent node whose specific type is not needed for path construction. It primarily manages the INode pointer and its path.
