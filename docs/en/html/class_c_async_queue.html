Detailed Description

template<typename TItem>
class CAsyncQueue< TItem >

A concrete implementation of IAsyncQueue<TItem>. This class provides a thread-safe queue using a std::list for item storage and iridium::threading::Synchronized<std::mutex> for synchronization. It is non-copyable.

Template Parameters
TItemThe type of items to be stored in the queue.
