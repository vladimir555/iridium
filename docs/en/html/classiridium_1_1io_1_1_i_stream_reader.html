virtual Buffer::TSharedPtr iridium::io::IStreamReader::read ( size_t const & size = DEFAULT_BUFFER_SIZE)
pure virtual

Reads data from the stream.

Parameters
sizeThe maximum number of bytes to read. If 0, returns any previously accumulated buffer.
Returns
A shared pointer to a Buffer.
  • a valid Buffer with data if read was successful.
  • an empty Buffer if no data is currently available.
  • nullptr on EOF or if the stream was closed unexpectedly.
