Detailed Description

template<typename TValue>
class iridium::io::fs::FileValue< TValue >

A template class that represents a value persisted in a file. The value is loaded from the specified file upon construction. When a new value is assigned to a FileValue object, it updates its internal state and writes the new value back to the file. Conversions between TValue and std::string (for file I/O) are handled by the iridium::convertion::convert framework.

Template Parameters
TValueThe type of the value to be stored and managed. This type must have appropriate iridium::convertion::convert specializations to and from std::string.

Constructs a FileValue object, loading its initial value from a file. Attempts to read the content of the specified file, converts it to TValue, and stores it. If the file does not exist, cannot be opened, or if conversion fails, the default_value is used.

Parameters
file_nameThe path to the file from which to load and to which to save the value.
default_valueThe value to use if the file cannot be read or its content is invalid. Defaults to a default-constructed TValue if not provided.
Exceptions
std::runtime_errorif file reading or initial conversion from string to TValue fails catastrophically (though some errors might result in using default_value).
