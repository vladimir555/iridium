Detailed Description

Concrete implementation of a file stream providing read, write, and seeking capabilities. This class implements IFileStreamReader and IFileStreamWriter (and by extension IFileStream and IStream). It uses C standard I/O functions (FILE*) for underlying file operations. The class is non-copyable.
