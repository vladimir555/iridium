Detailed Description

Interface for log event sinks, also known as appenders. Defines the contract for classes that consume log events and output them to a specific destination (e.g., console, file, network).

std::atomic< bool > iridium::logging::ISink::is_gmt_time
static

Global atomic flag indicating whether timestamps in log messages should be in GMT/UTC. If true, GMT/UTC time is used; otherwise, local time is used. This flag affects the output of makeLine.
