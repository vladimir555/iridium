Detailed Description

A file-backed cache implementation for strings. This class implements the ICache interface, providing a mechanism to store strings in memory with the ability to load from and persist (flush) to a specified file. Strings are typically accessed by an ID, which in this implementation corresponds to their index in an internal vector.
