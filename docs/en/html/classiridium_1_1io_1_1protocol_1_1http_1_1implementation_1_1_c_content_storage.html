Retrieves content as a readable file stream from the configured root path. The requested path is appended to the m_root_path to locate the file.

Parameters
pathThe relative path to the content within the root directory.
Returns
A shared pointer to an IFileStreamReader for the requested file. Returns nullptr if the file does not exist or cannot be opened (e.g., due to permissions).
