Detailed Description

template<typename TSubject>
class iridium::pattern::IPrototype< TSubject >

Interface for the Prototype design pattern. This pattern allows for the creation of new objects by copying an existing object (the prototype), without being dependent on their concrete classes. The TSubject template parameter typically refers to the base class or interface of the objects that are being cloned, ensuring that the clone method returns a pointer of that general type.

Template Parameters
TSubjectThe type of the object that will be cloned. This is often the class itself that implements this interface, or a base class/interface type in a hierarchy.
