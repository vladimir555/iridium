Определения типов

typedef INodeType< std::string > iridium::parsing::INode
 Определяет INode как специализацию интерфейса INodeType с std::string в качестве типа значения. Обычно представляет собой обобщенный узел в разобранной структуре данных (например, конфигурационный файл или документ XML/JSON), где каждый узел может иметь имя, атрибуты, дочерние узлы и строковое значение.
