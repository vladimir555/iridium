class  iridium::logging::implementation::CSinkFile
 Конкретная реализация приемника для вывода сообщений журнала в файл, с поддержкой ротации файлов. Этот приемник записывает отформатированные сообщения журнала по указанному пути к файлу и может ротировать лог-файл на основе стратегий, таких как ежедневная ротация или ротация при каждой инициализации.
