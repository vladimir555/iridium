Подробное описание

Класс Buffer для работы с сырыми двоичными данными, унаследованный от std::vector<uint8_t>.

Предоставляет различные конструкторы для создания буферов из строк или других буферов, а также вспомогательные функции для манипулирования данными, например, для проверки суффиксов.
