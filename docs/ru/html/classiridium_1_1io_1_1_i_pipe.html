Класс iridium::io::IPipe

Интерфейс, представляющий канал, объединяющий возможности доступа к потокам и передачи данных. Канал представляет собой средство для потока данных, позволяющее чтение и запись, и может активно передавать данные. Этот интерфейс агрегирует IPipeStreams для доступа к конечным точкам потока и IPipeTransmitter для операций потока данных. Подробнее...

Подробное описание

Интерфейс, представляющий канал, объединяющий возможности доступа к потокам и передачи данных. Канал представляет собой средство для потока данных, позволяющее чтение и запись, и может активно передавать данные. Этот интерфейс агрегирует IPipeStreams для доступа к конечным точкам потока и IPipeTransmitter для операций потока данных.
