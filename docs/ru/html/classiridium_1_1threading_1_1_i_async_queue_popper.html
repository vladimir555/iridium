Методы

◆ pop() [1/2]

template<typename TItem>
virtual TItems iridium::threading::IAsyncQueuePopper< TItem >::pop ( bool const & is_do_wait = true)
pure virtual

Извлекает элементы из очереди. Извлекает один или несколько элементов из начала очереди. Поведение относительно ожидания зависит от реализации и флага is_do_wait.

Аргументы
is_do_waitЕсли true, метод может блокироваться до появления элементов. Если false, он может немедленно вернуться, если очередь пуста.
Возвращает
Список элементов, извлеченных из очереди. Список может быть пустым в зависимости от is_do_wait и состояния очереди.

Замещается в CAsyncQueue< TItem >, CWorker< TInputItem, TOutputItem >, CWorkerPool< TInputItem, TOutputItem >, iridium::threading::implementation::CAsyncQueue< TItem >, iridium::threading::implementation::CWorker< TInputItem, TOutputItem >, iridium::threading::implementation::CWorkerPool< TInputItem, TOutputItem >, iridium::threading::implementation::CWorkerPoolPopper< TItem > и iridium::threading::implementation::CWorkerPopper< TItem >.

◆ pop() [2/2]

template<typename TItem>
virtual TItems iridium::threading::IAsyncQueuePopper< TItem >::pop ( std::chrono::nanoseconds const & timeout)
pure virtual

Извлекает элементы из очереди с тайм-аутом. Пытается извлечь элементы из начала очереди, ожидая в течение указанного периода времени.

Аргументы
timeoutМаксимальная продолжительность ожидания появления элементов.
Возвращает
Список элементов, извлеченных из очереди. Список может быть пустым, если элементы не появятся в течение тайм-аута.

Замещается в CAsyncQueue< TItem >, CWorker< TInputItem, TOutputItem >, CWorkerPool< TInputItem, TOutputItem >, iridium::threading::implementation::CAsyncQueue< TItem >, iridium::threading::implementation::CWorker< TInputItem, TOutputItem >, iridium::threading::implementation::CWorkerPool< TInputItem, TOutputItem >, iridium::threading::implementation::CWorkerPoolPopper< TItem > и iridium::threading::implementation::CWorkerPopper< TItem >.
