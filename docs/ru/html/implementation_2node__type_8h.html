class  iridium::parsing::implementation::CNodeType< TValue >
 Конкретная реализация интерфейса INodeType. Этот шаблон класса предоставляет полную реализацию для узла в древовидной структуре. Он управляет именем узла (используя интернирование строк для эффективности), его основным значением типа TValue и списком дочерних узлов. Класс спроектирован для поддержки клонирования через интерфейс IPrototype. Подробнее...
