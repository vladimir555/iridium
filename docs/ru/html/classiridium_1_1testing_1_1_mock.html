Шаблон класса iridium::testing::Mock< TClass >

Базовый класс для создания мок-объектов. Для создания мока для интерфейса или класса TClass, наследуйте от Mock<TClass>. Этот класс предоставляет механизм для перехвата вызовов методов и определения поведения моков. Подробнее...

template<typename TClass>
class iridium::testing::Mock< TClass >

Базовый класс для создания мок-объектов. Для создания мока для интерфейса или класса TClass, наследуйте от Mock<TClass>. Этот класс предоставляет механизм для перехвата вызовов методов и определения поведения моков.

◆ create() [1/2]

template<typename TClass>
template<typename ... TArgs>
static ::std::shared_ptr< TClass > iridium::testing::Mock< TClass >::create ( TArgs &&... args)

Статический фабричный метод для создания экземпляров. Если мокирование активно (m_is_mocked истинно), пытается вернуть предварительно зарегистрированный мок-объект. В противном случае создает новый экземпляр TClass с помощью std::make_shared.

Параметры шаблона
TArgsВариативные шаблонные аргументы для конструктора TClass при создании реального объекта.
Аргументы
argsАргументы для передачи в конструктор TClass.
Возвращает
Разделяемый указатель на TClass (либо мок, либо реальный экземпляр).
Исключения
std::runtime_errorесли `m_is_mocked` истинно, но мок-объекты не зарегистрированы.
