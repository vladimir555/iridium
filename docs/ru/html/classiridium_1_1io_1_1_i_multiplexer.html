Подробное описание

Интерфейс для мультиплексора ввода-вывода.

Мультиплексор ввода-вывода отслеживает несколько потоков ввода-вывода (например, сокеты, файловые дескрипторы) и ожидает, пока один или несколько из них станут готовы для выполнения операций ввода-вывода (например, чтения или записи). Это центральный компонент в масштабируемых, неблокирующих архитектурах ввода-вывода.
