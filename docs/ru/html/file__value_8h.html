class  iridium::io::fs::FileValue< TValue >
 Шаблонный класс, представляющий значение, сохраняемое в файле. Значение загружается из указанного файла при конструировании. Когда объекту FileValue присваивается новое значение, он обновляет свое внутреннее состояние и записывает новое значение обратно в файл.
