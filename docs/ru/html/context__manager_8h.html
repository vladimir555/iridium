class  iridium::io::IContextManager
 Интерфейс для управления коллекцией контекстов ввода-вывода (IContext). Менеджер контекстов отвечает за создание, уничтожение и, возможно, распределение операций ввода-вывода или потоков между несколькими контекстами. Это может использоваться для управления ресурсами, балансировки нагрузки или обработки различных типов задач ввода-вывода в отдельных контекстах. Подробнее...
