Функции

template<typename T>
std::string iridium::convertion::getTypeName ()
 Предоставляет доступ к функции, которая возвращает строковое представление имени типа. Эта функция импортируется в пространство имен iridium::convertion из внутренней реализации. Обычно используется как шаблонная функция, например, iridium::convertion::getTypeName<MyType>(). Точный характер возвращаемого имени (например, является ли оно деманглированным, включает ли полное пространство имен и т.д.) зависит от базовой реализации в implementation/type_name.h.
