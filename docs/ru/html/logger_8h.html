Классы

class  iridium::logging::Logger
 Центральный механизм логирования для приложения. Этот класс реализован как потокобезопасный singleton и управляет коллекцией приемников логов на основе предоставленной конфигурации. Он отвечает за отправку событий журнала соответствующим приемникам. Подробнее...
 
struct  iridium::logging::LogStreamDummy
 Потоковый объект-пустышка для логирования. Эта структура используется для уровней логирования, которые в данный момент отключены. Ее operator<< ничего не делает, фактически отбрасывая любые данные, направленные в поток. Подробнее...
 
struct  iridium::logging::LogStream
 Представляет активный поток для конструирования сообщения журнала. Объект этого типа обычно создается макросами логирования (например, LOGI). Он накапливает данные через operator<< и при уничтожении отправляет завершенное событие журнала в Logger singleton. Подробнее...
