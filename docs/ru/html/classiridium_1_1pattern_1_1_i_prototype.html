Подробное описание

template<typename TSubject>
class iridium::pattern::IPrototype< TSubject >

Интерфейс для шаблона проектирования "Прототип". Этот шаблон позволяет создавать новые объекты путем копирования существующего объекта (прототипа), не завися от их конкретных классов. Параметр шаблона TSubject обычно ссылается на базовый класс или интерфейс клонируемых объектов, гарантируя, что метод клонирования возвращает указатель этого общего типа.

Параметры шаблона
TSubjectТип объекта, который будет клонирован. Часто это сам класс, реализующий этот интерфейс, или тип базового класса/интерфейса в иерархии.
