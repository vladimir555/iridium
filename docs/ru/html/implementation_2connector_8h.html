class  iridium::db::implementation::CConnector
 Базовый класс для реализаций коннекторов баз данных. Этот класс предоставляет общую функциональность, такую как хранение конфигурации базы данных, и гарантирует, что объекты коннекторов не могут быть скопированы. Предполагается, что конкретные коннекторы баз данных (например, для MySQL, PostgreSQL) будут наследоваться от этого класса. Подробнее...
