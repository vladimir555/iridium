class  iridium::io::fs::ICache
 Интерфейс для кэша, который хранит строки и извлекает их с использованием числовых идентификаторов. Этот интерфейс определяет операции для добавления строк в кэш (что возвращает идентификатор), извлечения строк по идентификатору, удаления записей, очистки кэша и проверки его состояния. Наследуется от IInitializable, что предполагает, что экземпляры кэша могут требовать фазы инициализации. Учитывая его расположение в пространстве имен fs, он может быть предназначен для кэширования строк файлов, путей или других строковых данных, связанных с файлами. Подробнее...
