Подробное описание

Интерфейс для управления коллекцией контекстов ввода-вывода (IContext). Менеджер контекстов отвечает за создание, уничтожение и, возможно, распределение операций ввода-вывода или потоков между несколькими контекстами. Это может использоваться для управления ресурсами, балансировки нагрузки или обработки различных типов задач ввода-вывода в отдельных контекстах.

virtual IContext::TSharedPtr iridium::io::IContextManager::acquireContext ( Event::TSharedPtr const & event,
IMultiplexer::TSharedPtr const & multiplexer )
pure virtual

Получает или выбирает подходящий контекст ввода-вывода для обработки данного события, возможно, связывая его с определенным мультиплексором ввода-вывода. Это может использоваться в сценарии, когда события поступают и должны быть направлены в подходящий контекст для обработки.

Аргументы
eventУмный указатель на Event, которое необходимо обработать.
multiplexerУмный указатель на IMultiplexer, который может быть связан с контекстом или использоваться им.
Возвращает
Умный указатель на IContext, считающийся подходящим для обработки события. Может вернуть nullptr или выбросить исключение, если подходящий контекст не может быть получен.

Замещается в iridium::io::implementation::CContextManager.
