Подробное описание

template<typename TNodeView>
class iridium::parsing::serialization::NodeViewPtr< TNodeView >

Обертка для лениво инициализируемого разделяемого указателя на TNodeView. Используется для опциональных или рекурсивных определений узлов в структурах данных, определенных макросами сериализации (например, DEFINE_NODE_PTR). Фактический TNodeView обычно создается только при вызове get() и если существует базовый узел.

Параметры шаблона
TNodeViewТип NodeView, которым управляет эта обертка указателя.

template<typename TNodeView>
std::shared_ptr< TNodeView > iridium::parsing::serialization::NodeViewPtr< TNodeView >::get ( ) const

Получает разделяемый указатель на TNodeView. Если TNodeView еще не был создан (например, потому что базовый узел не существовал при конструировании), вернет нулевой указатель.

Возвращает
Разделяемый указатель на TNodeView или nullptr, если он недоступен/не инициализирован.
