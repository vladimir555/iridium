inline

Получает строку с подробной информацией о сборке и окружении. Эта функция собирает различные определения препроцессора и свойства системы для создания отчета о среде сборки, включая компилятор, архитектуру, стандарт C++ и реализацию STL. Информация генерируется при первом вызове и сохраняется в статическом буфере.

Возвращает
C-строка с информацией о сборке. Строка находится в статическом буфере и не должна изменяться или освобождаться. Последующие вызовы возвращают указатель на тот же буфер.
