class  iridium::parsing::implementation::CXMLParser
 Конкретная реализация парсера для данных в формате XML (eXtensible Markup Language). Этот класс предоставляет функциональность для разбора строки XML в структурированное дерево INode, представляя элементы XML, атрибуты и текстовое содержимое.
