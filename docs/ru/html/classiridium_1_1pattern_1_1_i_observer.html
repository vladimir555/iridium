Подробное описание

template<typename TEvent>
class iridium::pattern::IObserver< TEvent >

Интерфейс для 'Наблюдателя' в рамках шаблона проектирования "Наблюдатель". Объекты, реализующие этот интерфейс, могут подписываться на экземпляры IObservable для получения уведомлений о событиях типа TEvent. Когда происходит событие, наблюдаемый объект вызовет метод handle своих подписанных наблюдателей.

Параметры шаблона
TEventТип данных события, для обработки которого предназначен этот наблюдатель.

pure virtual

Определяет общие элементы интерфейса для IObserver, такие как typedef-ы умных указателей (TSharedPtr, TWeakPtr и т.д.).

Обрабатывает событие, полученное от IObservable, на который подписан этот наблюдатель. Этот метод вызывается IObservable при возникновении соответствующего события.

Аргументы
eventДанные события типа TEvent, которые должны быть обработаны наблюдателем.
