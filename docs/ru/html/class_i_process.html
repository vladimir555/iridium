Подробное описание

Интерфейс для взаимодействия с системным процессом. Определяет методы для получения состояния процесса и, наследуя от iridium::io::IStreamPort, ожидается, что он предоставит доступ к стандартным потокам ввода-вывода процесса (stdin, stdout, stderr).
