Подробное описание

template<typename TReceiver>
class iridium::pattern::Broadcaster< TReceiver >

Реализует паттерн "Вещатель", позволяя отправлять сообщения нескольким получателям.

Этот класс управляет коллекцией получателей и предоставляет механизм для вызова определенной функции-члена у всех из них.
