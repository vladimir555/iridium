Класс iridium::parsing::serialization::NodeView< void >

Специализация NodeView для безузловых узлов (тип значения void). Обычно используется как базовый класс для структурированных узлов, определенных DEFINE_NODE_BEGIN или DEFINE_ROOT_NODE_BEGIN, или для представления родительского узла, конкретный тип которого не нужен для построения пути. В основном управляет указателем INode и его путем. Подробнее...

#include <node.h>

Подробное описание

Специализация NodeView для безузловых узлов (тип значения void). Обычно используется как базовый класс для структурированных узлов, определенных DEFINE_NODE_BEGIN или DEFINE_ROOT_NODE_BEGIN, или для представления родительского узла, конкретный тип которого не нужен для построения пути. В основном управляет указателем INode и его путем.
