Класс iridium::io::fs::implementation::CFileStream

Конкретная реализация файлового потока, предоставляющая возможности чтения, записи и позиционирования. Этот класс реализует IFileStreamReader и IFileStreamWriter (и, соответственно, IFileStream и IStream). Он использует стандартные функции ввода-вывода C (FILE*) для базовых файловых операций. Класс является некопируемым. Подробнее...

#include <file_stream.h>

overridevirtual

Читает до size байт данных из файлового потока в буфер. Реализует метод IStreamReader::read.

Аргументы
sizeМаксимальное количество байт для чтения.
Возвращает
Умный указатель на Buffer, содержащий прочитанные данные. Возвращает nullptr при EOF, если не было прочитано ни одного байта, или в случае ошибки. Возвращает пустой буфер, если size равен 0 и нет предварительно накопленных данных (нетипично для файловых потоков).
Исключения
iridium::io::fs::Exception(или производный) при ошибках чтения, если не возвращается nullptr.

Замещает iridium::io::IStreamReader.
