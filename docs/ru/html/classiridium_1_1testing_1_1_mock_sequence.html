Подробное описание

template<typename TClassMock>
class iridium::testing::MockSequence< TClassMock >

Предварительное объявление для шаблонного класса MockSequence.

Управляет последовательностью ожидаемых вызовов методов на мок-объекте. Этот класс позволяет определить строгий порядок, в котором ожидаются вызовы методов, а также конкретные аргументы для этих вызовов.

Параметры шаблона
TClassMockТип мок-класса (например, MyInterfaceMock), для которого определяется последовательность.

Выбрасывает исключение при нарушении последовательности. Примечание: Проверки минимального/максимального количества вызовов, похоже, закомментированы в исходном коде.

Аргументы
