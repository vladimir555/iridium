Класс iridium::io::fs::IFileStreamReader

Интерфейс для читаемых файловых потоков. Объединяет специфичные для файлов свойства потока из IFileStream с общими возможностями чтения потока из IStreamReader. Конкретные реализации этого интерфейса будут предоставлять фактическую логику для чтения данных из определенного файлового источника. Подробнее...
