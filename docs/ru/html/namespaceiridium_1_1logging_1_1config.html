Определяет корневую структуру для конфигурации логгера. Эта структура инкапсулирует все настройки для системы логирования, включая глобальные параметры и список конфигураций приемников (sinks).

Указывает, использовать ли время GMT (UTC) для временных меток логов. Если false, используется локальное время.

Уровень логирования по умолчанию для логгера. Приемники могут переопределять этот уровень.

Определяет список приемников логов, каждый со своей специфической конфигурацией.

Перечисление для поддерживаемых типов приемников логирования.

Тип этого приемника логирования (например, CONSOLE, FILE).

Указывает, работает ли этот приемник в асинхронном режиме.

URI для конфигурации приемника. Для файловых приемников это обычно путь к файлу. Для других типов может отличаться (например, строка подключения).

Специфический уровень логирования для этого приемника. Если установлено значение UNKNOWN, он наследует уровень логгера по умолчанию. В противном случае, это значение переопределяет уровень по умолчанию для данного приемника.

Создает конфигурацию логгера по умолчанию. Эта конфигурация обычно включает консольный приемник с уровнем логирования по умолчанию.

Возвращает
Структура TLogger, заполненная настройками по умолчанию.
