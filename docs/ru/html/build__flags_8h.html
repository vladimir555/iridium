Подробное описание

Определяет флаги препроцессора для условной компиляции. Эти флаги устанавливаются системой сборки (например, CMake) для включения/отключения функций, таких как поддержка баз данных или OpenSSL. Определения закомментированы, так как они должны быть определены извне конфигурацией сборки.
