class  iridium::io::fs::implementation::CFileStream
 Конкретная реализация файлового потока, предоставляющая возможности чтения, записи и позиционирования. Этот класс реализует IFileStreamReader и IFileStreamWriter (и, соответственно, IFileStream и IStream). Он использует стандартные функции ввода-вывода C (FILE*) для базовых файловых операций.
