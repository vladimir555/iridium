Класс iridium::io::IStreamabstract

Базовый интерфейс для потоков ввода-вывода. Предоставляет общие функциональные возможности для всех типов потоков, такие как доступ к базовым системным дескрипторам и получение связанного URI. Он также наследуется от IInitializable, подразумевая, что потоки могут требовать шага инициализации. Этот интерфейс использует виртуальное наследование, чтобы позволить производным интерфейсам, таким как IStreamPort, наследовать от нескольких интерфейсов, связанных с потоками (IStreamReader, IStreamWriter), без неоднозначности. Подробное описание

Базовый интерфейс для потоков ввода-вывода. Предоставляет общие функциональные возможности для всех типов потоков, такие как доступ к базовым системным дескрипторам и получение связанного URI. Он также наследуется от IInitializable, подразумевая, что потоки могут требовать шага инициализации. Этот интерфейс использует виртуальное наследование, чтобы позволить производным интерфейсам, таким как IStreamPort, наследовать от нескольких интерфейсов, связанных с потоками (IStreamReader, IStreamWriter), без неоднозначности.

◆ getHandles()

virtual std::list< uintptr_t > iridium::io::IStream::getHandles ( ) const
pure virtual

Макрос, используемый для определения общих элементов интерфейса (например, виртуального деструктора).

Получает список базовых дескрипторов операционной системы, связанных с этим потоком. Например, для сокетного потока это может вернуть файловый дескриптор сокета. Для других типов потоков это могут быть файловые дескрипторы или другие специфичные для ОС идентификаторы.

Возвращает
Список uintptr_t, представляющих дескрипторы ОС. Список может быть пустым, если дескрипторы отсутствуют.

Замещается в iridium::io::fs::implementation::CFileStream, iridium::io::implementation::CStreamBuffer и iridium::io::implementation::CStreamReaderList.

virtual URI::TSharedPtr iridium::io::IStream::getURI ( ) const

Получает URI, связанный с этим потоком, если таковой имеется. Может использоваться для идентификации источника или назначения потока (например, путь к файлу, сетевой адрес).

Возвращает
Умный указатель на объект URI или nullptr, если с потоком не связан URI.

Замещается в iridium::io::fs::implementation::CFileStream, iridium::io::implementation::CStreamBuffer и iridium::io::implementation::CStreamReaderList.
