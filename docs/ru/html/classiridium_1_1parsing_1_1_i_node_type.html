Подробное описание

template<typename TValue>
class iridium::parsing::INodeType< TValue >

Обобщенный интерфейс для узла в древовидной структуре данных.

Этот шаблонный класс определяет общие операции для узла, такие как доступ к его имени, значению и дочерним элементам. Он спроектирован для клонирования через интерфейс IPrototype.
