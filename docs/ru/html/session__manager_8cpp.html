template<typename TObject, typename ... TArgs>
std::list< typename TObject::TSharedPtr > iridium::createObjects ( size_t const & count,
TArgs ... args )

Создает список разделяемых указателей на объекты типа TObject.

Параметры шаблона
TObjectТип создаваемых объектов. Должен иметь статический метод create() и typedef TSharedPtr.
TArgsВариативный пакет шаблонных параметров для аргументов TObject::create().
Аргументы
countКоличество создаваемых объектов.
argsАргументы, которые будут переданы в метод TObject::create() для каждого объекта.
Возвращает
std::list из TObject::TSharedPtr на вновь созданные объекты.
