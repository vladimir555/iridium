config::TDatebase iridium::db::makeConfig (io::URI const &uri)
 Создает объект конфигурации базы данных из URI. Разбирает данный URI и извлекает параметры подключения к базе данных (такие как тип, хост, порт, пользователь, пароль, имя базы данных) для заполнения объекта config::TDatebase.
 
IConnector::TSharedPtr iridium::db::createConnector (config::TDatebase const &config)
 Создает экземпляр коннектора базы данных на основе предоставленной конфигурации. Эта функция действует как фабрика для создания конкретной реализации IConnector (например, коннектора MySQL, PostgreSQL), подходящей для данной конфигурации.
 
IConnector::TSharedPtr iridium::db::createConnector (io::URI const &uri)
 Создает экземпляр коннектора базы данных из URI. Это вспомогательная функция, которая сначала вызывает makeConfig(uri) для создания объекта конфигурации из URI, а затем вызывает createConnector(config) с этой конфигурацией.
