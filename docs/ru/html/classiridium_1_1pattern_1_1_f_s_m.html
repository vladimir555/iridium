Подробное описание

template<typename TState, typename TEvent>
class iridium::pattern::FSM< TState, TEvent >

Обобщенная конечная машина состояний (FSM).

Этот класс предоставляет простую реализацию FSM, где состояния и события являются шаблонными типами.
