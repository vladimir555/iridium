◆ read()

Buffer::TSharedPtr iridium::io::implementation::CStreamReaderBuffer::read ( size_t const & size)
overridevirtual

Читает данные из потока.

Аргументы
sizeМаксимальное количество байт для чтения. Если 0, возвращает ранее накопленный буфер.
Возвращает
Умный указатель на буфер.
  • корректный буфер с данными в случае успешного чтения.
  • пустой буфер, если данные в данный момент отсутствуют.
  • nullptr при достижении конца файла (EOF) или при неожиданном закрытии потока.

Замещает iridium::io::IStreamReader.
