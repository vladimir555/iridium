class  iridium::io::fs::implementation::CFileStreamWriter
 Конкретная реализация файлового потока, специализированная для записи. Этот класс наследуется от CFileStream и настраивает его для операций записи. Он предоставляет все функциональные возможности CFileStream (который реализует IFileStreamReader и IFileStreamWriter), но его конструктор позволяет указать, следует ли перезаписывать (усекать) файл или использовать режим записи по умолчанию (который может добавлять или усекать в зависимости от поведения CFileStream::TOpenMode::WRITE).
