class  iridium::io::implementation::CContext
 Конкретная реализация IContext и IPipeManager. Этот класс управляет состоянием и событиями для контекста обмена данными, обычно связанного с одним основным потоком (например, клиентским подключением). Он обрабатывает постановку событий в очередь, обработку событий на основе протокола и управление именованными каналами (pipes) для мультиплексирования данных через основной поток.
