Подробное описание

Базовый интерфейс для файловых потоков. Расширяет общий интерфейс IStream специфичными для файлов операциями, такими как получение статуса файла. Использует виртуальное наследование для поддержки множественного наследования в комбинированных интерфейсах, таких как IFileStreamWriter или IFileStreamReader.

pure virtual

Макрос, используемый для определения общих элементов интерфейса (например, виртуального деструктора).

Получает информацию о состоянии файла, связанного с этим потоком.

Возвращает
Структура TFileStatus, содержащая такие детали, как время последнего изменения и размер файла.
Исключения
iridium::Exceptionили производные классы, если статус файла не может быть получен.

Замещается в iridium::io::fs::implementation::CFileStream.
