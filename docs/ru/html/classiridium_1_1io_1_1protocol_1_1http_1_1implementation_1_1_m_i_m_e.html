Класс iridium::io::protocol::http::implementation::MIME

Класс-одиночка для управления и поиска MIME-типов на основе расширений файлов. Он предварительно загружает список распространенных MIME-типов и связанных с ними расширений. Класс предназначен только для чтения после инициализации, поэтому он "не синхронизирован". Подробнее...

◆ getByFileNameExtension()

std::string iridium::io::protocol::http::implementation::MIME::getByFileNameExtension ( std::string const & extension) const

Получает строку MIME-типа для заданного расширения файла.

Аргументы
extensionРасширение файла (например, "txt", "html", "jpg") без ведущей точки. Регистронезависимый.
Возвращает
Строка соответствующего MIME-типа (например, "text/plain", "image/jpeg"). Возвращает "application/octet-stream", если расширение не найдено.
