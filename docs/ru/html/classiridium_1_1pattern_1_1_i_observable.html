Интерфейс для 'Наблюдаемого объекта' (или 'Субъекта') в рамках шаблона проектирования "Наблюдатель". Объекты, реализующие этот интерфейс, могут быть отслеживаемы экземплярами IObserver. Когда происходит событие типа TEvent, наблюдаемый объект отвечает за уведомление своих подписанных наблюдателей.

Параметры шаблона
TEventТип данных события, о котором будут уведомляться наблюдатели. Этот параметр определяет контекст уведомлений.

◆ subscribe()

template<typename TEvent>
virtual void iridium::pattern::IObservable< TEvent >::subscribe ( typename IObserver< TEvent >::TWeakPtr const & observer)
pure virtual

Определяет общие элементы интерфейса для IObservable, такие как typedef-ы умных указателей (TSharedPtr, TWeakPtr и т.д.).

Подписывает наблюдателя на этот наблюдаемый объект. После подписки наблюдатель будет получать уведомления о событиях типа TEvent, происходящих в этом наблюдаемом объекте.

Аргументы
observerРазделяемый указатель на экземпляр IObserver<TEvent>, который желает получать обновления.
Исключения
std::logic_errorМожет быть выброшено, если наблюдатель уже подписан или если подписка невозможна по какой-либо причине.

Замещается в iridium::pattern::implementation::CObservable< TEvent >.

Отписывает наблюдателя от этого наблюдаемого объекта. Наблюдатель больше не будет получать уведомления от этого наблюдаемого объекта.

Аргументы
observerРазделяемый указатель на экземпляр IObserver<TEvent>, который должен быть отписан.
Исключения
std::logic_errorМожет быть выброшено, если наблюдатель не был ранее подписан или не может быть найден.

Замещается в iridium::pattern::implementation::CObservable< TEvent >.
