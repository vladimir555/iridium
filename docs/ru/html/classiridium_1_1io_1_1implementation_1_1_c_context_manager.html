Класс iridium::io::implementation::CContextManager

Конкретная реализация интерфейса IContextManager. Управляет жизненным циклом множества экземпляров IContext, связывая их с потоками и обрабатывая их создание, получение, освобождение и удаление. Использует мьютекс для потокобезопасных операций со своими внутренними коллекциями. Подробнее...

Подробное описание

Конкретная реализация интерфейса IContextManager. Управляет жизненным циклом множества экземпляров IContext, связывая их с потоками и обрабатывая их создание, получение, освобождение и удаление. Использует мьютекс для потокобезопасных операций со своими внутренними коллекциями.

Получает контекст, связанный с потоком события, потенциально создавая его, если он не существует. Если контекст для потока события существует, он возвращается. В противном случае, и если событие является Event::NEW_STREAM, создается новый контекст с использованием потока и протокола события. Полученный контекст помечается как "полученный" для предотвращения одновременного изменения или преждевременного удаления.

Аргументы
eventСобытие, поток которого используется для поиска или создания контекста.
multiplexerМультиплексор для связи с потоком, если создается новый контекст и поток нуждается в управлении.
Возвращает
Умный указатель на полученный или вновь созданный контекст, или nullptr, если контекст не удалось связать или создать.
