Файл socket.h

Определяет реализации сокетов для платформы Linux. Поведение этого файла зависит от макроса BUILD_FLAG_OPENSSL. Если BUILD_FLAG_OPENSSL определен, предполагается, что он предоставляет OpenSSL-специфичные классы сокетов (хотя эта функциональность в значительной степени закомментирована в текущей версии). Подробное описание

Определяет реализации сокетов для платформы Linux. Поведение этого файла зависит от макроса BUILD_FLAG_OPENSSL. Если BUILD_FLAG_OPENSSL определен, предполагается, что он предоставляет OpenSSL-специфичные классы сокетов (хотя эта функциональность в значительной степени закомментирована в текущей версии). В противном случае, он по умолчанию использует typedef для общих реализаций сокетов Unix (unix::CSocketPeer, unix::CSocketClient, unix::CSocketAcceptor).
