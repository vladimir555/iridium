Класс iridium::parsing::implementation::CXMLParser

Конкретная реализация парсера для данных в формате XML (eXtensible Markup Language). Этот класс предоставляет функциональность для разбора строки XML в структурированное дерево INode, представляя элементы XML, атрибуты и текстовое содержимое. Подробное описание

Конкретная реализация парсера для данных в формате XML (eXtensible Markup Language). Этот класс предоставляет функциональность для разбора строки XML в структурированное дерево INode, представляя элементы XML, атрибуты и текстовое содержимое. Он также может формировать (компоновать) дерево INode обратно в его строковый формат XML.

Формирует (компонует) древовидную структуру узлов, представляющую данные XML, в строку XML.

Аргументы
root_nodeКонстантный разделяемый указатель на корневой узел INode структуры XML для компоновки.
Возвращает
Строка, содержащая представление дерева узлов в формате XML.
Исключения
std::runtime_errorМожет быть выброшено, если структура узлов недействительна для компоновки в строку XML (например, недопустимые символы XML в именах/значениях без надлежащей обработки реализацией).

Замещает iridium::parsing::IParser.

◆ parse()

INode::TSharedPtr iridium::parsing::implementation::CXMLParser::parse ( std::string const & source) const
overridevirtual

Разбирает строку XML в древовидную структуру узлов. Результирующее дерево представляет элементы XML как узлы, с атрибутами и текстовым содержимым, доступными через интерфейс узла.

Аргументы
sourceСтрока XML для разбора.
Возвращает
Разделяемый указатель на корневой узел INode разобранной структуры XML.
Исключения
std::runtime_errorМожет быть выброшено, если строка XML некорректна или происходит ошибка разбора.

Замещает iridium::parsing::IParser.
