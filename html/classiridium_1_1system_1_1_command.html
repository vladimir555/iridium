Класс iridium::system::Command

Представляет команду оболочки для выполнения в виде внешнего процесса. Этот класс предоставляет функциональность для формирования команды из пути к приложению и аргументов, запуска внешнего процесса, управления временем его выполнения с настраиваемым тайм-аутом, а также получения его статуса завершения и стандартного вывода. Использует базовый IProcess и IMultiplexer для событийно-ориентированного ввода-вывода и управления тайм-аутом. Подробнее...

Подробное описание

Представляет команду оболочки для выполнения в виде внешнего процесса. Этот класс предоставляет функциональность для формирования команды из пути к приложению и аргументов, запуска внешнего процесса, управления временем его выполнения с настраиваемым тайм-аутом, а также получения его статуса завершения и стандартного вывода. Использует базовый IProcess и IMultiplexer для событийно-ориентированного ввода-вывода и управления тайм-аутом.
