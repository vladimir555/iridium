◆ createConnector() [1/3]

IConnector::TSharedPtr iridium::db::createConnector ( config::TDatebase const & config)

Создает экземпляр коннектора базы данных на основе предоставленной конфигурации. Эта функция действует как фабрика для создания конкретной реализации IConnector (например, коннектора MySQL, PostgreSQL), подходящей для данной конфигурации.

Аргументы
configОбъект конфигурации базы данных (config::TDatebase), указывающий параметры подключения. Поле Type в конфигурации определяет, какой тип коннектора будет создан.
Возвращает
Умный указатель (IConnector::TSharedPtr) на экземпляр IConnector. Возвращает нулевой указатель или выбрасывает исключение, если коннектор не может быть создан для данной конфигурации.
Исключения
iridium::db::Exceptionили другие исключения при ошибке создания коннектора.

◆ createConnector() [2/3]

IConnector::TSharedPtr iridium::db::createConnector ( io::URI const & uri)

Создает экземпляр коннектора базы данных из URI. Это вспомогательная функция, которая сначала вызывает makeConfig(uri) для создания объекта конфигурации из URI, а затем вызывает createConnector(config) с этой конфигурацией.

Аргументы
uriURI, содержащий информацию о подключении к базе данных.
Возвращает
Умный указатель (IConnector::TSharedPtr) на экземпляр IConnector. Возвращает нулевой указатель или выбрасывает исключение, если коннектор не может быть создан.
Исключения
iridium::db::Exceptionили другие исключения, если разбор URI или создание коннектора не удалось.

◆ makeConfig()

config::TDatebase iridium::db::makeConfig ( io::URI const & uri)

Создает объект конфигурации базы данных из URI. Разбирает данный URI и извлекает параметры подключения к базе данных (такие как тип, хост, порт, пользователь, пароль, имя базы данных) для заполнения объекта config::TDatebase.

Аргументы
uriURI, содержащий информацию о подключении к базе данных. Схема URI может определять тип базы данных (например, "mysql://", "postgres://").
Возвращает
Объект config::TDatebase, заполненный параметрами из URI.
Исключения
iridium::db::Exceptionили другие исключения, если разбор URI не удался или схема не поддерживается.
