#define DEFINE_FSM_MAP_TRANSITION(Event, StateFrom, StateTo)
 Вспомогательный макрос для определения структуры TTransition, обычно для инициализации списка переходов для CFSM. Предполагается, что класс, в котором используется этот макрос, имеет шаблонный метод handle<Event, StateFrom, StateTo>(), который будет служить обработчиком перехода. Этот обработчик привязывается к текущему экземпляру (this) с помощью std::bind.

Вспомогательный макрос для определения структуры TTransition, обычно для инициализации списка переходов для CFSM. Предполагается, что класс, в котором используется этот макрос, имеет шаблонный метод handle<Event, StateFrom, StateTo>(), который будет служить обработчиком перехода. Этот обработчик привязывается к текущему экземпляру (this) с помощью std::bind.

Аргументы
EventКонкретное значение события, которое вызывает этот переход.
StateFromЗначение исходного состояния, из которого происходит этот переход.
StateToЗначение целевого состояния, в которое КА переходит после этого перехода.
