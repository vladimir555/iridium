Класс iridium::pattern::implementation::CInitializer

Конкретная реализация IInitializable, управляющая коллекцией других объектов IInitializable. Этот класс позволяет группировать несколько инициализируемых компонентов и управлять их жизненным циклом (инициализация и финализация) как единым целым. Он гарантирует, что все управляемые объекты инициализируются в порядке их предоставления и финализируются в обратном порядке. Подробнее...

Подробное описание

Конкретная реализация IInitializable, управляющая коллекцией других объектов IInitializable. Этот класс позволяет группировать несколько инициализируемых компонентов и управлять их жизненным циклом (инициализация и финализация) как единым целым. Он гарантирует, что все управляемые объекты инициализируются в порядке их предоставления и финализируются в обратном порядке.

Конструктор(ы)

◆ CInitializer()

template<typename ... TArgs>
iridium::pattern::implementation::CInitializer::CInitializer ( TArgs const &... items)

Определяет стандартные вспомогательные средства реализации для CInitializer, такие как статический метод create для создания экземпляров через разделяемые указатели.

Конструирует CInitializer с переменным количеством элементов IInitializable. Элементы сохраняются внутри и будут инициализированы и финализированы этим экземпляром CInitializer.

Параметры шаблона
TArgsВариативный пакет шаблонных параметров, где каждый тип в TArgs должен быть преобразуем в IInitializable::TSharedPtr.
Аргументы
itemsПакет параметров разделяемых указателей на объекты IInitializable.
Исключения
std::runtime_errorесли какой-либо из предоставленных `items` является nullptr.

Инициализирует список m_items предоставленным пакетом разделяемых указателей на объекты IInitializable. Итерирует по элементам, чтобы убедиться, что ни один из них не является null, выбрасывая std::runtime_error при обнаружении null-элемента.
