Структура iridium::logging::LogStream

Представляет активный поток для конструирования сообщения журнала. Объект этого типа обычно создается макросами логирования (например, LOGI). Он накапливает данные через operator<< и при уничтожении отправляет завершенное событие журнала в Logger singleton. Подробнее...

#include <logger.h>

Подробное описание

Представляет активный поток для конструирования сообщения журнала. Объект этого типа обычно создается макросами логирования (например, LOGI). Он накапливает данные через operator<< и при уничтожении отправляет завершенное событие журнала в Logger singleton.
