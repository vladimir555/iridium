Подробное описание

template<typename TReceiver>
class iridium::pattern::Broadcaster< TReceiver >

Шаблон класса, реализующий роль вещателя (или субъекта) в шаблоне, подобном "Наблюдателю". Управляет коллекцией объектов-получателей (наблюдателей) и предоставляет механизм для трансляции вызова определенного метода всем зарегистрированным получателям. Это позволяет отделить вещателя от его получателей.

Параметры шаблона
TReceiverТип объектов-получателей. Этот тип должен предоставлять typedef TSharedPtr (например, typedef std::shared_ptr<MyReceiverType> TSharedPtr;) для своего типа разделяемого указателя, который используется для управления экземплярами получателей.

void iridium::pattern::Broadcaster< TReceiver >::attach ( typename TReceiver::TSharedPtr const & receiver)

Присоединяет получателя к этому вещателю. После этого получатель будет уведомляться о последующих трансляциях.

Аргументы
receiverРазделяемый указатель на объект-получатель для присоединения.
Исключения
std::logic_errorесли получатель уже присоединен (дубликат).

◆ broadcast()

template<typename TReceiver>

void iridium::pattern::Broadcaster< TReceiver >::broadcast ( void(TReceiver_::* f )(TArgs const &...),
TArgs const &... args )

Транслирует вызов определенной функции-члена каждому присоединенному получателю. У всех присоединенных получателей, совместимых с TReceiver_, будет вызвана их функция-член f с предоставленными аргументами.

Параметры шаблона
TReceiver_Фактический тип получателя, у которого существует функция-член. Должен быть TReceiver или тип, производный/совместимый с ним.
TArgsВариативный пакет шаблонных параметров для аргументов функции-члена.
Аргументы
fУказатель на функцию-член, которая будет вызвана у каждого получателя.
argsАргументы, которые будут переданы в функцию-член.
