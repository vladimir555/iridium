Подробное описание

Конкретная реализация IContentStorage, которая обслуживает контент из локального каталога файловой системы.

◆ getContent()

fs::IFileStreamReader::TSharedPtr iridium::io::protocol::http::implementation::CContentStorage::getContent ( std::string const & path)
overridevirtual

Получает контент в виде читаемого файлового потока из настроенного корневого пути. Запрошенный path добавляется к m_root_path для определения местоположения файла.

Аргументы
pathОтносительный путь к контенту в корневом каталоге.
Возвращает
Умный указатель на IFileStreamReader для запрошенного файла. Возвращает nullptr, если файл не существует или не может быть открыт (например, из-за прав доступа).

Замещает iridium::io::protocol::http::IContentStorage.
