 connector.h
 Определяет базовый класс CConnector для конкретных реализаций коннекторов баз данных. Этот заголовочный файл предоставляет общую функциональность для коннекторов баз данных, например, хранение конфигурации.
 
 mysql_connector.cpp
 
 mysql_connector.h
 Определяет класс CMySQLConnector, конкретную реализацию IConnector для баз данных MySQL. Этот коннектор использует MySQL C API. Содержимое компилируется только если определен BUILD_FLAG_MYSQL.
 
 postgres_connector.cpp
 
 postgres_connector.h
 Определяет класс CPostgresConnector, конкретную реализацию IConnector для баз данных PostgreSQL. Этот коннектор использует C-библиотеку libpq. Содержимое компилируется только если определен BUILD_FLAG_POSTGRES.
