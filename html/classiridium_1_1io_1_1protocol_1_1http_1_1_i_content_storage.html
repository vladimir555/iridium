Подробное описание

Интерфейс для системы хранения контента, используемый для получения HTTP-контента (например, файлов) по указанному пути.

Методы

◆ getContent()

virtual fs::IFileStreamReader::TSharedPtr iridium::io::protocol::http::IContentStorage::getContent ( std::string const & path)
pure virtual

Получает контент в виде читаемого файлового потока из хранилища.

Аргументы
pathПуть к контенту (например, относительный путь к файлу для веб-сервера).
Возвращает
Умный указатель на IFileStreamReader для доступа к контенту. Возвращает nullptr или выбрасывает исключение, если контент не найден или к нему нет доступа.

Замещается в iridium::io::protocol::http::implementation::CContentStorage.
