Шаблон класса iridium::parsing::serialization::NodeViewList< TNodeView >

Представляет список объектов NodeView определенного типа TNodeView. Этот класс предоставляет интерфейс для итерации и изменения списка сложных дочерних узлов или списка атрибутов в структуре данных, определенной макросами сериализации. Подробнее...

Подробное описание

template<typename TNodeView>
class iridium::parsing::serialization::NodeViewList< TNodeView >

Представляет список объектов NodeView определенного типа TNodeView. Этот класс предоставляет интерфейс для итерации и изменения списка сложных дочерних узлов или списка атрибутов в структуре данных, определенной макросами сериализации.

Параметры шаблона
TNodeViewТип NodeView, который содержит этот список. Обычно это структура, сгенерированная DEFINE_NODE_LIST_BEGIN или DEFINE_ATTRIBUTE_LIST.
