Файл socket.h

Определяет базовую реализацию сокета для платформы macOS. Данная реализация в настоящее время использует typedef CSocket для общей Unix-реализации CSocketBase, используя POSIX API сокетов, доступные на macOS. Более специфичные типы сокетов (клиентские, серверные, равноправные узлы) для macOS обычно наследуются отсюда. Подробнее...

#include "../unix/socket.h"
