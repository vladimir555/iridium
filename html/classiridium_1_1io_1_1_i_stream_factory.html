Подробное описание

Интерфейс для фабрики потоков, отвечающей за создание различных типов портов потоков ввода-вывода. Эта фабрика специализируется на создании клиентских и серверных сокетных портов потоков на основе конфигураций URI. Конкретные реализации этого интерфейса будут обрабатывать специфику создания сокетов для различных сетевых протоколов или транспортных уровней, указанных в URI.

Методы

◆ createClientSocket()

virtual IStreamPort::TSharedPtr iridium::io::IStreamFactory::createClientSocket ( URI const & uri)
pure virtual

Макрос, используемый для определения общих элементов интерфейса (например, виртуального деструктора).

Создает клиентский сокетный порт потока, настроенный для подключения к указанному URI. URI должен содержать необходимую информацию (например, имя хоста, порт, протокол, такой как tcp или udp) для установления клиентского сетевого соединения. Возвращенный порт потока затем может использоваться для двунаправленной связи.

Аргументы
uriURI, указывающий адрес и параметры для подключения клиентского сокета.
Возвращает
Умный указатель на IStreamPort, представляющий клиентский сокет. Возвращает nullptr или выбрасывает исключение, если клиентский сокет не может быть создан или настроен.
Исключения
iridium::Exceptionили производные классы в случае сбоя (например, неверный URI, сетевые ошибки).

virtual IStreamPort::TSharedPtr iridium::io::IStreamFactory::createServerSocket ( URI const & uri)
pure virtual

Создает серверный сокетный порт потока, настроенный для прослушивания на адресе, указанном в URI. URI должен содержать необходимую информацию (например, локальный IP-адрес, порт, протокол) для привязки прослушивающего сокета. Этот порт потока обычно используется сервером для приема входящих клиентских соединений.

Аргументы
uriURI, указывающий локальный адрес и параметры для серверного сокета.
Возвращает
Умный указатель на IStreamPort, представляющий серверный прослушивающий сокет. Возвращает nullptr или выбрасывает исключение, если серверный сокет не может быть создан или настроен.
Исключения
iridium::Exceptionили производные классы в случае сбоя (например, неверный URI, адрес уже используется, сетевые ошибки).
