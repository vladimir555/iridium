Класс iridium::parsing::implementation::CHTTPParser

Конкретная реализация парсера для HTTP-сообщений. Этот класс предоставляет функциональность для разбора необработанных строк HTTP-запросов или ответов в структурированное дерево INode, а также для формирования (компоновки) дерева INode, представляющего HTTP-сообщение, обратно в его необработанный строковый формат. ◆ parse()

INode::TSharedPtr iridium::parsing::implementation::CHTTPParser::parse ( std::string const & source) const
overridevirtual

Разбирает строку HTTP-сообщения в древовидную структуру узлов. Результирующее дерево обычно представляет заголовки, тело и другие компоненты HTTP в виде узлов.

Аргументы
sourceНеобработанная строка HTTP-сообщения (запрос или ответ) для разбора.
Возвращает
Разделяемый указатель на корневой узел INode разобранной структуры HTTP-сообщения.
Исключения
std::runtime_errorМожет быть выброшено, если HTTP-сообщение некорректно или происходит ошибка разбора.

Замещает iridium::parsing::IParser.
