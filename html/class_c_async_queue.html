Шаблон класса CAsyncQueue< TItem >

Конкретная реализация IAsyncQueue<TItem>. Этот класс предоставляет потокобезопасную очередь, используя std::list для хранения элементов и iridium::threading::Synchronized<std::mutex> для синхронизации. Класс не является копируемым. Подробнее...

template<typename TItem>
class CAsyncQueue< TItem >

Конкретная реализация IAsyncQueue<TItem>. Этот класс предоставляет потокобезопасную очередь, используя std::list для хранения элементов и iridium::threading::Synchronized<std::mutex> для синхронизации. Класс не является копируемым.

TItemТип элементов, сохраняемых в очереди.
