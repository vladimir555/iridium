Подробное описание

template<typename TEvent, typename TState>
class iridium::pattern::implementation::CFSM< TEvent, TState >

Конкретная реализация интерфейса IFSM, использующая список переходов (поведение, подобное карте). Этот Конечный Автомат управляет состояниями и переходами на основе предопределенного списка объектов TTransition. Он поддерживает связывание обработчиков событий с конкретными переходами и может вести историю переходов.

Параметры шаблона
TEventТип, представляющий события, которые могут вызывать переходы состояний. Должен быть сравним на равенство.
TStateТип, представляющий состояния КА. Должен быть сравним на равенство. Обычно это перечисление (enum).

◆ doAction()

template<typename TEvent, typename TState>
TState iridium::pattern::implementation::CFSM< TEvent, TState >::doAction ( TEvent const & event)
overridevirtual

Обрабатывает событие, пытается найти соответствующий переход и обновляет состояние КА. Если найден допустимый переход (соответствующий текущему состоянию и событию), выполняется связанный обработчик (если есть), состояние обновляется, и переход записывается в историю.

Аргументы
eventСобытие для обработки.
Возвращает
Новое состояние КА после обработки события.
Исключения
std::runtime_errorесли не найден переход для текущего состояния и данного события.

Замещает iridium::pattern::IFSM< TEvent, TState >.
