Класс URI

Представляет и облегчает разбор универсальных идентификаторов ресурсов (URI). Этот класс может разбирать строку URI на ее составные компоненты, такие как схема (протокол), информация о пользователе, хост, порт, путь и аргументы запроса. Он предоставляет методы для доступа к этим компонентам. Подробнее...

#include <uri.h>

Подробное описание

Представляет и облегчает разбор универсальных идентификаторов ресурсов (URI). Этот класс может разбирать строку URI на ее составные компоненты, такие как схема (протокол), информация о пользователе, хост, порт, путь и аргументы запроса. Он предоставляет методы для доступа к этим компонентам.
