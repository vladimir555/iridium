Подробное описание

Интерфейс для записываемых файловых потоков. Объединяет специфичные для файлов свойства потока из IFileStream с общими возможностями записи потока из IStreamWriter.

Методы

◆ flush()

virtual void iridium::io::fs::IFileStreamWriter::flush ( )
pure virtual

Макрос, используемый для определения общих элементов интерфейса.

Сбрасывает все буферизованные данные в базовый файл. Гарантирует, что все данные, записанные в поток через write(), физически сохранены в файле.

Исключения
iridium::Exceptionили производные классы, если операция сброса не удалась.

Замещается в iridium::io::fs::implementation::CFileStream.
