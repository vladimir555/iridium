Подробное описание

template<typename TItem>
class iridium::pattern::IContainer< TItem >

Обобщенный интерфейс для контейнера, который может хранить элементы определенного типа. Этот интерфейс определяет базовые операции для добавления элемента в контейнер и удаления элемента из контейнера. Конкретные реализации будут предоставлять специфический механизм хранения и логику управления элементами.

Параметры шаблона
TItemТип элементов, которые будут храниться в контейнере.

Добавляет элемент в контейнер.

Аргументы
itemЭлемент для добавления в контейнер.
Исключения
