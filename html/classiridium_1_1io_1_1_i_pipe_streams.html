Подробное описание

Интерфейс для доступа к потокам чтения и записи каналоподобной структуры. Предоставляет методы для получения и установки базовых потоковых компонентов, которые обрабатывают чтение и запись.

Устанавливает как читателя, так и писателя потока для этого канала. Этот метод позволяет настраивать или перенастраивать базовые потоки, используемые для операций с каналом.

Аргументы

writerУмный указатель на IStreamWriter, который будет использоваться для записи.
