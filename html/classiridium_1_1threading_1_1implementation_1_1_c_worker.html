Подробное описание

template<typename TInputItem, typename TOutputItem = TInputItem>
class iridium::threading::implementation::CWorker< TInputItem, TOutputItem >

Реализует IWorker для обработки элементов из входной очереди и помещения результатов в выходную очередь, используя выделенный рабочий поток.

Параметры шаблона
TInputItemТип элементов во входной очереди.
TOutputItemТип элементов в выходной очереди (по умолчанию TInputItem).
