Подробное описание

Интерфейс для объекта потока. Предоставляет базу для реализаций потоков, включая инициализацию, именование и сравнение. Наследует от IInitializable, предполагая, что потоки могут иметь явные фазы настройки/завершения.

Оператор сравнения для упорядочивания объектов IThread, например, для использования в отсортированных контейнерах. Обычно сравнивает на основе идентификатора потока или другого уникального идентификатора.

threadДругой объект IThread для сравнения.

True, если этот поток должен быть упорядочен перед другим потоком, иначе false.
